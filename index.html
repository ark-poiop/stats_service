시작하기

  1. CSV 또는 Excel 파일을 업로드하거나 샘플 데이터를 사용하세요.
  2. 분석하고 싶은 방법을 선택하세요.
  3. 독립변수와 종속변수를 선택하세요.
  4. 분석 실행 버튼을 클릭하세요.
  5. 결과를 표, 그래프, 요약으로 확인하세요.

분석 방법

  • 기술통계: 데이터의 기본적인 특성을 파악
  • 가설검정: 통계적 유의성을 검증
  • 상관분석: 변수 간의 관계를 분석
  • 회귀분석: 변수 간의 인과관계를 분석

결과 해석

  • p-값: 0.05 미만이면 통계적으로 유의함
  • R²: 1에 가까울수록 모델의 설명력이 높음
  • 상관계수: -1~1 사이의 값으로 관계의 강도와 방향을 나타냄
